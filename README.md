# Discordbot

A bot just for fun

### Requirements:
#### Ubuntu:
```bash
ffmpeg
python3
python-is-python3
python3-venv
```
## Usage
Clone repo to final destination:
```bash
git clone https://github.com/Linus13000/Discordbot
```
Open folder:
```bash
cd Discordbot
```
Setup:
```bash
./build.sh
```
Run:
```bash
./start.sh
```
